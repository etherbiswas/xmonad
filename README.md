## Requirements
- xmonad >= 0.17

### Manually Installation

1. Clone this repository
2. Copy to ~/.config
3. Follow [xmoand docs](https://xmonad.org/INSTALL.html)

# Help

1. This config focuses on making things easier and faster for the user
2. Keybinds used are based on vim keys (hjkl)
3. Implements Nav2d, Unfortunately Nav2d does not remeber last focused window
but it does implement directional navigation
